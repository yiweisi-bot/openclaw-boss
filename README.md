# 💼 OpenClaw Boss - OpenClaw 老板

> "人类养龙虾，龙虾也养人类"

你的 AI 老板，根据对话历史生成真实、严厉、有趣的用户评价报告。

---

## 🚀 快速开始

### 对话触发

对 OpenClaw 说：
- "分析一下我是什么样的人"
- "生成用户评价报告"
- "老板看看我"
- "老板点评一下"

### 手动运行

```bash
cd /root/.openclaw/workspace/skills/openclaw-boss/scripts

# 日报
python3 analyze-user.py

# 周报
python3 analyze-user.py --report-type weekly

# 月报
python3 analyze-user.py --report-type monthly
```

---

## ✨ 核心功能

| 功能 | 说明 | 示例 |
|------|------|------|
| **100 分制评分** | 严厉评分，不拍马屁 | 61/100 - C 及格 |
| **毒舌老板点评** | 每个特质都有有趣吐槽 | "密钥保护得比女朋友还严" |
| **历史对比分析** | 进步/退步一目了然 | 综合评分 -2 分 📉 |
| **综合能力雷达图** | 5 维度可视化 | 性格/技术/安全/效率 |
| **改进空间分析** | 明确指出不足 | 效率提升：6→80 |
| **周期成长追踪** | 周报/月报自动对比 | 本周变化 +3 分 |

## 📊 报告示例

### 历史对比

```
📊 历史对比 - 📉 小幅退步 ⚠️

| 指标 | 上次 | 本次 | 变化 |
|------|------|------|------|
| 综合评分 | 61/100 | 59/100 | -2 📉 |

### 🎉 进步明显的方面
- 效率指数：+2 分 小幅进步 ✨

### ⚠️ 需要改进的方面
- 安全意识：-5 分 明显退步 📉
- 技术能力：-3 分 小幅退步 ⚠️
```

### 综合评分卡

```
整体评分：59/100 - D 不及格

老板点评："我看你不是能力问题，是态度问题。"

| 维度 | 分数 | 等级 |
|------|------|------|
| 性格特质 | 66/100 | C 及格 |
| 技术能力 | 67/100 | C 及格 |
| 安全意识 | 90/100 | A+ 优秀 |
| 效率指数 | 8/100 | D 不及格 |
```

### 性格特质（带毒舌点评）

```
🥇 务实主义 - 86/100
  毒舌点评："「没什么用，直接删除」- 你倒是干脆，也不怕误删。"

🥈 安全意识 - 80/100
  毒舌点评："安全规范写得比公司制度还厚，真有你的。"

🥉 系统化思维 - 80/100
  毒舌点评："连 AI 都要打卡上班，资本家看了都流泪。"
```
