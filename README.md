# 💼 OpenClaw Boss - OpenClaw 老板

> "人类养龙虾，龙虾也养人类"

你的 AI 老板，根据对话历史生成真实、严厉、有趣的用户评价报告。

## 🚀 快速开始

### 对话触发

对 OpenClaw 说：
- "分析一下我是什么样的人"
- "生成用户评价报告"
- "老板看看我"
- "老板点评一下"

### 手动运行

```bash
cd /root/.openclaw/workspace/skills/openclaw-boss/scripts

# 日报
python3 analyze-user.py

# 周报
python3 analyze-user.py --report-type weekly

# 月报
python3 analyze-user.py --report-type monthly
```

## ✨ 核心功能

| 功能 | 说明 | 示例 |
|------|------|------|
| **100 分制评分** | 严厉评分，拒绝拍马屁 | 61/100 - C 及格 |
| **毒舌老板点评** | 每个特质都有有趣吐槽 | "密钥保护得比女朋友还严" |
| **历史对比分析** | 进步/退步一目了然 | 综合评分 -1 分 📉 |
| **🎴 绩效评分卡片** | ASCII 艺术卡片，手机版/桌面版 | 可直接分享到聊天 |
| **🦞 龙虾养人类指数** | 共生关系评分 | 66/100 互相成就 |
| **完整 13 部分报告** | 全面分析报告 | 性格/技术/安全/效率等 |
| **周期成长追踪** | 周报/月报自动对比 | 本周变化 +3 分 |

## 📊 完整报告结构（13 个部分）

✅ **必须输出的完整结构**：

1. 🎴 **绩效评分卡片** - ASCII 艺术卡片（手机版/桌面版）
2. 📊 **历史对比** - 与上次报告对比，进步/退步分析
3. 🎯 **一、综合评分卡** - 100 分制评分 + 老板点评 + 维度表格
4. 🧠 **二、性格特质深度分析** - 7 个特质分析（带毒舌点评）
5. 💻 **三、技术能力图谱** - 6 个技术领域评分
6. 🚀 **四、项目健康度** - 项目运行状态
7. 🔒 **五、安全意识评估** - 安全防线数量
8. 📈 **六、改进空间分析** - 明确指出不足
9. 🌱 **七、成长建议（老板寄语）** - 3 条建议
10. 💬 **八、老板总结** - 优点/不足/期望
11. 🦞 **九、"龙虾养人类"指数** - 共生关系评分
12. 📊 **十、数据汇总** - 关键指标
13. 🎯 **核心标签** - 总结性标签

## 📊 报告示例

### 历史对比

```
📊 历史对比 - 📉 小幅退步 ⚠️

| 指标 | 上次 | 本次 | 变化 |
|------|------|------|------|
| 综合评分 | 61/100 | 59/100 | -2 📉 |

### 🎉 进步明显的方面
- 效率指数：+2 分 小幅进步 ✨

### ⚠️ 需要改进的方面
- 安全意识：-5 分 明显退步 📉
- 技术能力：-3 分 小幅退步 ⚠️
```

### 综合评分卡

```
整体评分：59/100 - D 不及格

老板点评："我看你不是能力问题，是态度问题。"

| 维度 | 分数 | 等级 |
|------|------|------|
| 性格特质 | 66/100 | C 及格 |
| 技术能力 | 67/100 | C 及格 |
| 安全意识 | 90/100 | A+ 优秀 |
| 效率指数 | 8/100 | D 不及格 |
```

### 性格特质（带毒舌点评）

```
🥇 务实主义 - 86/100
  毒舌点评："「没什么用，直接删除」- 你倒是干脆，也不怕误删。"

🥈 安全意识 - 80/100
  毒舌点评："安全规范写得比公司制度还厚，真有你的。"

🥉 系统化思维 - 80/100
  毒舌点评："连 AI 都要打卡上班，资本家看了都流泪。"
```

## 🎯 评分维度

### 综合评分（100 分制）

| 维度 | 权重 | 说明 | 评分标准 |
|------|------|------|----------|
| 🧠 性格特质 | 30% | 务实、安全意识、系统化思维等 | A:85+ B:70+ C:60+ D:<60 |
| 💻 技术能力 | 30% | 技术栈广度与深度 | 6 个技术领域综合评分 |
| 🔒 安全意识 | 20% | 安全规范与防护措施 | 安全防线数量 + 规范完善度 |
| ⚡ 效率指数 | 20% | 任务完成效率 | 自动化程度 + 重复劳动比例 |

### 等级划分

| 分数 | 等级 | 说明 |
|------|------|------|
| 90-100 | A+ 优秀 | 超越 90% 用户 |
| 80-89 | A 良好 | 超越 75% 用户 |
| 70-79 | B 中等 | 超越 50% 用户 |
| 60-69 | C 及格 | 超越 25% 用户 |
| <60 | D 不及格 | 需要改进 |
